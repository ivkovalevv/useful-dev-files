# Настройка VPN на собственном VPS

## 1. Обновляем пакеты
```bash
sudo apt update
```

## 2. Устанавливаем панель управления 3X-UI
```bash
bash <(curl -Ls https://raw.githubusercontent.com/mhsanaei/3x-ui/master/install.sh)
```
На этапе установки SSL-сертификата будет выбор
`"Choose an option (default 2 for IP):"
("Выберите опцию (по умолчанию 2 для IP):")`

Варианты:
1. Let's Encrypt для домена — если у вас есть доменное имя, привязанное к этому серверу. Сертификат будет действителен 90 дней и обновится автоматически.
2. Let's Encrypt для IP-адреса — если у вас нет домена, но есть белый (публичный) IP-адрес. Сертификат будет действителен всего 6 дней и тоже обновится автоматически. Этот вариант выбран по умолчанию.
3. Свой SSL-сертификат — если у вас уже есть готовые файлы сертификата и вы укажете к ним путь вручную.

В случае, если выбрали 3 вариант, необходимо будет указать доменное имя и пути к сертификату.
Они будут выглядеть так:
```bash
/etc/letsencrypt/live/your-site.ru/fullchain.pem
/etc/letsencrypt/live/your-site.ru/privkey.pem
```

В конце установки появится ссылка для входа в админ панель:
```bash
Access URL:  https://your-site.ru:45678/hG7reR53LfZ31cSaQp/
```

Порт `45678` должен быть открыт и подключение к нему должно быть разрешено.

Проверть правила файрвола:
```bash
sudo ufw status verbose
```

Открыть подключения к порту:
```bash
sudo ufw allow 45678/tcp
sudo ufw reload
```

## 3. Устанавливаем имя пользователя и пароль для админ-панели
```bash
x-ui
```

В меню есть пункт, который отвечает за сброс учетных данных. Обычно это пункт 6 — "Reset Username & Password"
Нажимаем `6`, затем `Enter` и вводим имя пользователя и пароль.

После этого можно будет зайти в админ панель.

## 4. Создание нового подключения в админ панели

Нажимаем `+ Создать подключение` и настраиваем его.

Указанный по умолчанию порт нужно будет открыть в фаерволе командой
```bash
sudo ufw allow 33552/tcp
sudo ufw reload
```

Безопасность выбираем `Reality` и в открывшемся окне генерируем ключи.

Остальные настройки оставлякм по умолчанию и нажимаем `Созранить`.

## 5. Получаем ссылку для клиента

В разделе "Подключения", рядом с id подключения нажимаем `+`
В информации о клиенте копируем URL

## 6. Подвключаемся

1. Скачиваем в App Store приложение `Streisand`
2. Нажимаем `+` для создания подключения и выбираем `Добавить из буфера`, чтобы вставить заранеескопированный URL из админ панели.
3. Подключаемся и проверяем доступность сервисов.
